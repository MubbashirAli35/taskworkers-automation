Run "run_paper_trade_workers.sh" to start all 9 paper_trade_workers.

ngctw1.sh .... ngctw9.sh are individual scripts to start respective trade workers.

Before running change the python file and conda environment names in ngctw1,sh, ... , ngctw9.sh

run_paper_trade_workers.sh will start 9 screens in background running each paper_trade_worker.
